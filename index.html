For - co wiemy

{% for user in users %}
	{{ user.username }}
{% endfor %}

Filtrowanie

{% for user in users|slice(0, 10) %}
	{{ user.username }}
{% endfor %}
{% for user in users if user.active %}
	{{ user.username }}
{% endfor %}

For - loop

Jak sprawdzić w którym miejscu pętli jesteśmy?

{% for user in users %}
	{{ loop.index0 }} - {{ user.username }}
{% endfor %}

Jak sprawdzić czy to już koniec?

{% for user in users %}
	{% if loop.last %} 
		To koniec
	{% endif %}
{% endfor %}

For - loop

Jak sprawdzić czy users naprawdę istnieją?

{% for user in users %}
	{{ loop.index0 }} - {{ user.username }}
{% else %}
	No users :(
{% endfor %}

Include - co wiemy

Jak zaincludować plik?

{% include 'parts/pagincja.twig' %}

Jak zaincludować plik z parametrami?

{% include('parts/pagincja.twig', { a: 'test' }) %}

Include

Jak zaincludować bez zmiennych z zewnątrz?

{% include('parts/pagincja.twig', { a: 'test' }, with_context = false) %}

Jak zainlcudować jeden spośród kliku plików?

{% include(['parts/pagincja-new.twig', 'parts/pagincja.twig']) %}

Jak nie pokazać błędu kiedy plik nie istnieje?

{% include(['parts/pagincja-new.twig', 'parts/pagincja.twig'], ignore_missing = false) %}
